# Big Pants Apps Landing Page

Static marketing site for Big Pants Apps.

## Pages

- `index.html`: studio homepage.
- `flash-sign.html`: Flash Sign Banner product page.

## Contact

Use `info@bigpantsapps.com` for support, launch questions, and general contact.

## Local Preview

Open `index.html` in a browser, or serve this folder with any static file server.

## Deployment

The current production server serves static files from:

```text
/home/ubuntu/bigpantsapps/current
```

For the simplest deployment, copy the contents of this folder to that server directory and reload nginx only when nginx config changes.
