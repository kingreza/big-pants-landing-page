Quick guide

Using Blackjack Strategy Trainer

Choose table rules

Use Settings to match the rule set you want to study, including decks, dealer behavior, surrender, and double-after-split options.

Start a focused drill

Use Basic Practice for mixed decisions, Pair Practice for splits, Soft Practice for soft totals, or Weakest Hands for missed and under-practiced spots.

Answer each hand

Choose Hit, Stand, Double, Split, or Surrender. When you miss, the app shows the best move for the active table rules.

Use the reference table

Open the strategy chart when you want to study hard totals, soft totals, and pairs without running a drill.

Review progress

Local stats and heatmaps help you find decisions that need more repetition. Progress stays on your device.

Training only

The app has no chips, betting rounds, bankroll mechanics, casino affiliates, ads, analytics, tracking, or real-money play.

Troubleshooting

Things to try first

The answer differs from another chart

Check the active table rules first. Basic strategy changes with rules like surrender, dealer soft 17, deck count, and double-after-split.

Weakest Hands feels limited

Complete more practice rounds. Weakest Hands becomes more useful after the app has enough local attempt history to identify patterns.

Progress is not on another device

Practice history is stored locally on the device. The app does not require accounts, analytics, cloud sync, or tracking.

Contact

Still need help?

Email info@bigpantsapps.com and include your device model, iOS version, app version if available, the active table rules, and a short description of what happened.
